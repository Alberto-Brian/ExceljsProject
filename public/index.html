🔗 Documentação da API

Endpoints para Gerar Excel:

GET /excel/basic - Gerar planilha básica
GET /excel/vendas - Gerar relatório de vendas
GET /excel/produtos - Gerar lista de produtos
POST /excel/custom - Gerar planilha personalizada

Endpoints de Informações:

GET /api/templates - Listar templates disponíveis
GET /api/downloads - Listar arquivos para download
GET /api/info - Informações do sistema
